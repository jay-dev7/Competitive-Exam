Quick Fire Shortcuts
⚡ Trick 1 — Factoring sin²+cos²

Whenever you see sin²θ + cos²θ anywhere in an expression → replace with 1 instantly. Most exam questions collapse to 1 at the end!

⚡ Trick 2 — sec²−tan² = 1

If you see sec²θ alone → write as 1 + tan²θ. If you see sec²θ − tan²θ → write 1 directly.

⚡ Trick 3 — Complementary shortcut

sin 70° = cos 20°, sin 55° = cos 35°. In exams, questions like sin 35° · sec 55° = sin 35° / cos 55° = sin 35° / sin 35° = 1.

⚡ Trick 4 — tan² identity trick

If given (sec θ − tan θ)(sec θ + tan θ) → always equals 1. Great for simplification questions!

⚡ Trick 5 — Value substitution

For questions like "if sinθ = 3/5, find cosθ": use the 3-4-5 triangle. cosθ = 4/5 immediately — no formula needed!

⚡ Trick 6 — sinθ + cosθ = k

If sinθ + cosθ = k, then squaring both sides: sin2θ = k² − 1. Saves time in indirect value questions.

⚡ Trick 7 — tanθ from sinθ

If sin θ = a/b, then tan θ = a / √(b²−a²). Draw a right triangle mentally — opposite = a, hypotenuse = b, adjacent = √(b²−a²).

⚡ Trick 8 — 45° shortcut

At θ = 45°: sin = cos = 1/√2, tan = cot = 1, sec = cosec = √2. Any expression that simplifies to θ = 45° becomes trivial!
