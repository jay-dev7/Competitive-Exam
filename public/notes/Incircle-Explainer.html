🔵 What is an Incircle?

Triangle sides
Incircle & r
Tangent points
Angle bisectors

An incircle is the largest circle that fits inside a triangle, touching all three sides exactly once.

  • The center is called the Incentre (I)
  • Incentre = intersection of all three angle bisectors
  • The inradius (r) = perpendicular distance from Incentre to any side
  • The incircle always lies inside the triangle
  • Three tangent points — one on each side
Core Formula
r = Area / s
s = (a + b + c) / 2  (semi-perimeter)
Area = r × s

📐 Incircle in a Right Triangle

Right angle at C • legs = a, b • hypotenuse = c
Special Formula — Right Triangle
r = (a + b − c) / 2

where:
a, b = two legs
c = hypotenuse

This formula works because in a right triangle, the incircle touches both legs and the hypotenuse. The geometry gives us a clean shortcut — no Heron's formula needed!

▶ Example: Legs = 6, 8. Find r.
Step 1: c = √(6² + 8²) = √100 = 10
Step 2: r = (6 + 8 − 10) / 2
= 4 / 2
∴ r = 2 cm ✅
▶ Example: Legs = 5, 12. Find r.
c = √(25 + 144) = √169 = 13
r = (5 + 12 − 13) / 2 = 4/2
∴ r = 2 cm ✅

🔺 Incircle in an Equilateral Triangle

All sides = a • Incentre = Centroid = Circumcentre
Equilateral Triangle Formulas
r = a / (2√3) = a√3 / 6
R = a / √3 = a√3 / 3

Key relation: R = 2r
Area = √3/4 × a²

In an equilateral triangle, the Incentre, Centroid, and Circumcentre are all the same point! The circumradius R is always twice the inradius r.

▶ Example: Side = 6 cm. Find r.
r = 6 / (2√3) = 3/√3
= 3√3/3 = √3
∴ r = √3 ≈ 1.73 cm ✅
▶ Example: r = 2 cm. Find side a.
r = a√3/6 → a = 6r/√3
a = 12/√3 = 12√3/3 = 4√3
∴ a = 4√3 ≈ 6.93 cm ✅

✏️ Solved Examples (Heron's Formula Method)

EXAMPLE 1 — General Triangle

▶ Sides: 5, 12, 13. Find r.
s = (5+12+13)/2 = 15
Area = √[15×10×3×2]
= √900 = 30
r = Area/s = 30/15
∴ r = 2 cm ✅

Note: 5-12-13 is a Pythagorean triple so r = (5+12-13)/2 = 2 also works!

EXAMPLE 2 — Given Perimeter & r

▶ Perimeter = 48 cm, r = 4 cm. Find Area.
s = 48/2 = 24
Area = r × s = 4 × 24
∴ Area = 96 cm² ✅

EXAMPLE 3 — Find r from Area and Perimeter

▶ Area = 84 cm², Perimeter = 42 cm. Find r.
s = 42/2 = 21
r = Area/s = 84/21
∴ r = 4 cm ✅

3-4-5 → r = 1

5-12-13 → r = 2
8-15-17 → r = 3

💡 Shortcut

If Perimeter and r are given:
→ Area = r × (P/2)

If Area and P are given:
→ r = (2 × Area) / P

🎯 Key Facts

• Incircle always inside triangle
• Incentre = intersection of angle bisectors
• Equilateral: Incentre = Centroid = Circumcentre
• r is perpendicular to all 3 sides

Quick Decision Tree

Given 3 sides? → Use Heron's → r = Area/s
Given right triangle legs? → r = (a+b−c)/2
Given equilateral side? → r = a√3/6
Given Perimeter + r? → Area = r × (P/2)
Given Area + Perimeter? → r = 2·Area/P
