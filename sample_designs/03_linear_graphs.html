📝 Forms of Linear Equation
General Formax + by + c = 0
Slope-Intercepty = mx + c
Point-Slopey−y₁ = m(x−x₁)
Two-Point Form(y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)
Intercept Formx/a + y/b = 1
Slope from generalm = −a/b
x-interceptset y=0 → x = −c/a
y-interceptset x=0 → y = −c/b
▶ 3x − 4y + 12 = 0. Find slope and intercepts.
m = −a/b = −3/(−4) = 3/4
x-int: y=0 → 3x=−12 → x=−4
y-int: x=0 → −4y=−12 → y=3
m=3/4, x-int=(−4,0), y-int=(0,3) ✓
📐 Writing Equations
Given slope m and y-intercept c
y = mx + c
Given slope m and point (x₁,y₁)
y − y₁ = m(x − x₁)
Given two points (x₁,y₁) and (x₂,y₂)
m = (y₂−y₁)/(x₂−x₁) then use point-slope

🔗 System Consistency Test
For a₁x+b₁y+c₁=0 and a₂x+b₂y+c₂=0
a₁/a₂ ≠ b₁/b₂ → Unique solution (intersect)
a₁/a₂ = b₁/b₂ = c₁/c₂ → Infinitely many (coincident)
a₁/a₂ = b₁/b₂ ≠ c₁/c₂ → No solution (parallel)

⚡ Trick 1 — Slope from General Form

From ax+by+c=0: slope = −a/b, x-int = −c/a, y-int = −c/b. No rearranging needed! 3 seconds per question.

⚡ Trick 2 — Parallel/Perp Test

Parallel: same slope m₁=m₂. Perpendicular: m₁×m₂=−1. If slopes are negative reciprocals of each other → perpendicular.

⚡ Trick 3 — Consistency Ratios

Compare a₁/a₂, b₁/b₂, c₁/c₂. All equal → same line. First two equal, third different → parallel (no solution). First two unequal → unique solution.

⚡ Trick 4 — Addition Method

When solving systems: if coefficients of one variable match → directly add or subtract. Fastest method. No fractions!

⚡ Trick 5 — Point on Line Test

Substitute (p,q) into equation. If result = 0 → on the line. Quick check for any MCQ option.

⚡ Trick 6 — Distance to Parallel Lines

For ax+by+c₁=0 and ax+by+c₂=0: distance = |c₁−c₂|/√(a²+b²). The 3-4-5 trick often applies: if a=3,b=4 → √(a²+b²)=5!
