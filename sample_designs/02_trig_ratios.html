🏔 Angle of Elevation & Depression
Key Formulas
tan θ = h/d → h = d·tan θ
sin θ = h/l → l = h/sin θ
cos θ = d/l → d = l·cos θ
Angle of depression = Angle of elevation
📐 Solved Examples
▶ Tower 30m. Elevation=30°. Distance?
tan30°=30/d → 1/√3=30/d
d=30√3≈51.96 m ✓
▶ Shadow=6√3 m, man=6m tall. Find sun elevation.
tanθ=6/6√3=1/√3
θ=30° ✓
▶ From 60m cliff, depression=60°. Dist from base?
tan60°=60/d → √3=60/d
d=60/√3=20√3 m ✓
📏 Two-Observer Problems
▶ Angles of elevation from A and B (100m apart) to top of tower are 30° and 60°. Find height.
h=d₁·tan30°=d₂·tan60°, d₁+d₂=100
d₁/√3=d₂·√3 → d₁=3d₂. 3d₂+d₂=100, d₂=25
h=25√3≈43.3 m ✓

⚡ Trick 1 — Pythagorean Triple First

Given any ratio like sin=3/5, immediately recognise 3-4-5 triple. All 6 ratios in 5 seconds. Memorise: 3-4-5, 5-12-13, 8-15-17, 7-24-25.

⚡ Trick 2 — sin²+cos²=1 everywhere

Spot sin²θ+cos²θ anywhere → replace with 1. Spot sec²−tan² → 1. Spot cosec²−cot² → 1. This alone solves 40% of identity MCQs!

⚡ Trick 3 — Complementary trick

sin 35°=cos 55°. Always. sin A / cos(90−A) = 1. Use this for sin²20+sin²70, tan1°×tan89° type questions instantly.

⚡ Trick 4 — H&D setup

Draw right triangle every time. Identify: observer → angle → object. tan θ = height / horizontal distance. Always start with tan for H&D problems.

⚡ Trick 5 — Product traps

If problem has cos0°×cos1°...×cos90° → answer is 0 (cos90°=0). If tan1°×...×tan89° → answer is 1. Check for these patterns first!

⚡ Trick 6 — sin+cosec=2 shortcut

If sinθ+cosecθ=2, then sinθ=1. Because AM≥GM → minimum of sinθ+1/sinθ is 2, achieved only when sinθ=1. So any power also = 1.
